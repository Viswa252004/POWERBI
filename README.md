Project Description

This project focuses on analyzing the content available on Amazon Prime Video using an interactive dashboard developed in Microsoft Power BI. The main objective of the project is to explore and visualize important information about movies and TV shows available on the platform.

The dataset includes details such as title, genre, rating, release year, director, and country. By analyzing this data, the dashboard provides insights into the total number of movies and TV shows, genre distribution, ratings classification, release year trends, and country-wise content availability.

Different visualizations such as bar charts, pie charts, and line charts are used to represent the data in a clear and understandable way. Interactive filters and slicers are also included to allow users to easily explore the dataset and gain meaningful insights.

This dashboard helps users quickly understand content trends and patterns in the Amazon Prime platform and demonstrates the use of data visualization techniques for effective data analysis.
